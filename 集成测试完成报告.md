### 成本计算
- **Prompt成本**: $0.003 per 1K tokens

## 发现的问题和修复

### 问题1: debugInfo未返回
**描述**: retrieveFromWebsite方法没有返回debugInfo字段

**修复**: 
```typescript
return {
  websiteUrl,
  status: 'success',
  keywordMatches: llmSearchResult.keywordMatches,
  documentResults: llmSearchResult.documentResults,
  retrievedAt: startTime,
  debugInfo: llmSearchResult.debugInfo, // 添加此行
};
```

**状态**: ✅ 已修复

### 问题2: Jest无法处理natural库的ES模块
**描述**: natural库使用ES模块，Jest默认配置无法处理

**修复**: 
1. 创建jest.config.js配置文件
2. 创建natural库的mock (src/__mocks__/natural.ts)
3. 配置moduleNameMapper

**状态**: ✅ 已修复

### 问题3: 测试内容长度不足
**描述**: 测试用的内容太短，被minChunkSize过滤掉

**修复**: 增加测试内容长度，确保超过100字符

**状态**: ✅ 已修复
