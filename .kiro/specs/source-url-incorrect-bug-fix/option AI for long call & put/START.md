# 期权交易助手 - 启动指南

## 🚀 快速启动

### 方式 1：运行演示程序（推荐）

```bash
node demo-simple.mjs
```

这将启动一个交互式命令行界面，您可以：
- 输入标的代码（如：AAPL）或中文名称（如：苹果）
- 分析市场情绪
- 查看推荐的期权合约
- 生成交易链接

### 方式 2：运行测试验证系统

```bash
# 运行所有测试
npm test

# 运行测试并查看覆盖率
npm run test:coverage
```

## 🎯 核心功能

### 1. 标的资产分析
- 支持股票代码输入（AAPL、TSLA、MSFT）
- 支持中文名称输入（苹果、特斯拉、微软）
- 多格式验证和智能建议

### 2. 市场情绪分析
- 技术指标分析（趋势、波动率、RSI）
- 支撑位和阻力位计算
- 智能策略推荐（Long Call / Long Put）

### 3. 期权合约筛选
- 基于策略的合约过滤
- Greeks 计算（Delta、Gamma、Theta、Vega）
- 智能评分和排序

### 4. 交易链接生成
- 一键生成交易链接
- 自选列表管理
- 批量操作支持

## 💡 提示

### 支持的命令
- **标的输入**：AAPL、苹果、TSLA、特斯拉、MSFT、微软
- **确认操作**：确认、是、好的、继续
- **拒绝操作**：重新选择、否、取消
- **特殊命令**：帮助、重新开始、退出

### 对话状态
系统使用状态机管理对话流程：
1. AWAITING_UNDERLYING - 等待标的输入
2. CONFIRMING_UNDERLYING - 确认标的
3. ANALYZING_UNDERLYING - 分析市场
4. SUGGESTING_STRATEGY - 建议策略
5. ANALYZING_OPTIONS - 分析期权
6. PRESENTING_CONTRACTS - 展示合约
7. AWAITING_SELECTION - 等待选择
8. GENERATING_TRADE_LINK - 生成链接
9. COMPLETED - 完成
