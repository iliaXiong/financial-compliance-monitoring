# Product Overview

Options Trading Tool - A conversational trading assistant focused on Long Call and Long Put option strategies.

## Core Purpose

Help traders analyze underlying assets, evaluate market sentiment, filter suitable option contracts, and provide convenient trading access through interactive dialogue.

## Target Users

Options traders who want intelligent guidance for Long Call (bullish) and Long Put (bearish) strategies.

## Key Workflows

1. Underlying asset selection and validation
2. Market sentiment analysis and strategy recommendation
3. Option contract filtering and ranking
4. Trade link generation and watchlist management

## Language

- User-facing content: Chinese (中文)
- Code and technical documentation: English
- Spec documents: Chinese
