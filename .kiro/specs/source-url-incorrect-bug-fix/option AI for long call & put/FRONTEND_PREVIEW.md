## 设计特点 Design Features

### 🎨 视觉设计
- **渐变背景**: 紫色到粉色的渐变（#667eea → #764ba2）
- **卡片式布局**: 白色圆角卡片，带阴影效果
- **图标系统**: Emoji 图标，直观易懂
- **颜色方案**: 
  - 主色：蓝色 (#2563eb)
  - 成功：绿色 (#10b981)
  - 警告：橙色 (#f59e0b)
  - 危险：红色 (#ef4444)

### 💬 交互设计
- **消息气泡**: 用户消息（紫色渐变）vs 助手消息（白色）
- **动画效果**: 消息滑入动画，按钮悬停效果
- **快速操作**: 每条消息下方的操作按钮
- **加载状态**: 旋转加载指示器

### 📱 响应式设计
- **桌面端**: 最大宽度 900px，居中显示
- **移动端**: 全屏显示，优化触摸操作
- **自适应网格**: 功能卡片自动调整布局

### ⌨️ 键盘支持
- **Enter**: 发送消息
- **Tab**: 在输入框和按钮间切换
- **Esc**: 清空输入（未来功能）

## 技术实现 Technical Implementation

### HTML 结构
- 语义化标签（header, main, footer）
- 无障碍属性（aria-labels）
- SEO 友好的元数据

### CSS 特性
- CSS 变量（主题定制）
- Flexbox 和 Grid 布局
- CSS 动画和过渡
- 自定义滚动条样式

### JavaScript 功能
- 事件驱动架构
- 状态机模式
- DOM 操作优化
- XSS 防护（HTML 转义）

## 浏览器预览 Browser Preview

在浏览器中访问 `http://localhost:3000` 后，您将看到：

1. **首次加载**: 显示欢迎卡片和快速开始按钮
2. **开始对话**: 点击快速按钮或输入标的代码
3. **实时交互**: 消息实时显示，带有平滑动画
4. **操作反馈**: 加载状态、按钮状态变化
5. **完整流程**: 从标的选择到交易链接生成

## 移动端适配 Mobile Adaptation

在移动设备上：
- 全屏显示，无边距
- 触摸优化的按钮大小
- 虚拟键盘自动弹出
- 滚动性能优化
