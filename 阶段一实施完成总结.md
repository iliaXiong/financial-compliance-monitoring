## 技术实现细节

### Chunk拆分策略
```typescript
// 1. 按段落拆分
paragraphs = content.split(/\n\n+/)

// 2. 长段落按句子拆分
if (paragraph.length > 500) {
  sentences = paragraph.split(/([。！？])/)
}

// 3. 过滤小于100字符的chunk
if (chunk.length >= 100) {
  chunks.push(chunk)
}
```

### BM25检索（基于TF-IDF）
```typescript
// 使用natural库的TfIdf类
tfidf.addDocument(chunk.content)
tfidf.tfidfs(query, (i, score) => {
  scores.push({ chunk, score })
})
```

### 优化后的Prompt格式
```
关键词: data retention

以下是可能相关的内容片段：

[Chunk 1 | https://example.com/page1]
内容1...

[Chunk 2 | https://example.com/page2]
内容2...

任务：
1. 判断关键词是否存在
2. 给出定义/解释/描述
3. 必须引用原文句子
4. 必须提供来源URL
5. 如果不存在，明确说不存在

输出格式（严格JSON）：
{
  "keyword": "...",
  "found": true/false,
  "definition": "...",
  "quotedSentence": "...",
  "sourceUrl": "...",
  "confidence": 0.0-1.0
}
```
