╔════════════════════════════════════════════════════════════════════╗
║                          数据流向                                   ║
╚════════════════════════════════════════════════════════════════════╝

1. 用户创建任务
   用户 → Vercel → Render → Supabase (保存任务)

2. 定时执行任务
   pg-boss (Supabase) → Render → 执行任务

3. 内容检索
   Render → Jina Reader API → 获取网页内容

4. AI 分析
   Render → LLM API → 生成总结

5. 保存结果
   Render → Supabase (保存结果)

6. 查看结果
   用户 → Vercel → Render → Supabase (读取结果)
