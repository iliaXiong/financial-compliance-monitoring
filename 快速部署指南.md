# ⚡ 快速部署指南

使用 Render + Supabase + Vercel 部署，完全免费！

## 🚀 一键部署

运行自动化脚本：

```bash
./scripts/deploy-render-supabase-vercel.sh
```

脚本会引导你完成所有配置步骤。

---

## 📝 手动部署（3 步）

### 第 1 步：Supabase 数据库 (5 分钟)

1. 访问 https://supabase.com，创建项目
2. 设置数据库密码，获取连接信息
3. 运行迁移：
   ```bash
   export DB_HOST=aws-0-ap-southeast-1.pooler.supabase.com
   export DB_PASSWORD=你的密码
   cd backend && npm install && npm run migrate
   ```

### 第 2 步：Render 后端 (10 分钟)

1. 访问 https://render.com，创建 Web Service
2. 配置：
   - Root Directory: `backend`
   - Build Command: `npm install && npm run build`
   - Start Command: `node dist/index.pgboss.js`
3. 添加环境变量（见下方）
4. 部署并获取 URL

**必需的环境变量**：
```bash
NODE_ENV=production
PORT=3000
DB_HOST=aws-0-ap-southeast-1.pooler.supabase.com
DB_PASSWORD=你的密码
JWT_SECRET=$(openssl rand -base64 32)

# LLM 配置（二选一）
# OpenAI:
OPENAI_API_KEY=your-key
OPENAI_MODEL=gpt-4

# 或自定义 API:
LLM_API_URL=your-url
LLM_API_KEY=your-key
LLM_MODEL=your-model
```

### 第 3 步：Vercel 前端 (5 分钟)

1. 创建 `frontend/.env.production`：
   ```bash
   VITE_API_BASE_URL=https://your-app.onrender.com
   ```

2. 部署：
   ```bash
   cd frontend
   vercel --prod
   ```

