#### 选项 2：Railway（推荐用于生产）

**优点**：
- $5 免费额度
- 24/7 运行
- 不需要信用卡（初期）

**步骤**：

1. **注册 Railway**：
   - 访问：https://railway.app
   - 用 GitHub 账号登录

2. **创建新项目**：
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择 `financial-compliance-monitoring`

3. **配置服务**：
   - Root Directory: `backend`
   - Build Command: `npm install && npm run build`
   - Start Command: `node dist/index.pgboss.js`

4. **添加环境变量**：
   在 Railway Dashboard 中添加所有环境变量（从 `.render-env.txt` 复制）

5. **获取 Railway URL**：
   部署完成后，Railway 会给你一个 URL，如：
   ```
   https://your-app.railway.app
   ```

6. **配置 Vercel**：
   在 Vercel 环境变量中设置：
   ```
   VITE_API_URL=https://your-app.railway.app
   ```
