## 核心功能

### 1. SimpleRetriever
```typescript
// Chunk拆分
const chunks = retriever.chunkContent(content, url, 'main');

// BM25检索
retriever.buildIndex(chunks);
const results = retriever.retrieve('关键词', 30);
```

**特性**:
- 智能拆分（100-500字符）
- BM25相关性计算
- Top-K检索
- 限制≤30 chunks

### 2. DebugLogger
```typescript
const logger = new DebugLogger('关键词', url);
logger.logChunking(chunks);
logger.logRetrieval(results);
logger.logLLMCall(tokens, cost, duration);
logger.validate(chunks);
const debugInfo = logger.getDebugInfo();
```

**收集信息**:
1. 内容抓取统计
2. Chunk拆分统计
3. 检索结果统计
4. LLM调用统计
5. LLM回答内容
6. 验证结果

### 3. 优化版LLM搜索
- ✅ 限制输入≤30 chunks
- ✅ 新prompt格式（强制引用）
- ✅ 返回置信度
- ✅ 完整debug信息
