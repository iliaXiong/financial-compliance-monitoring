## 技术实现

### Chunk拆分算法
```typescript
1. 按段落拆分 (\\n\\n)
2. 如果段落>500字符:
   - 按句子拆分 (。！？)
   - 组合句子直到接近500字符
3. 过滤<100字符的chunk
4. 保留元数据 (sourceUrl, pageType, chunkIndex)
```

### BM25检索（TF-IDF近似）
```typescript
1. 建立索引: tfidf.addDocument(chunk.content)
2. 计算相关性: tfidf.tfidfs(query, callback)
3. 排序: 按score降序
4. 返回Top-K
```

### LLM调用流程
```typescript
1. 检索Top 30 chunks
2. 构建优化后的prompt
3. 调用LLM API
4. 解析JSON响应
5. 验证引用句子和URL
6. 记录debug信息
```
