
## 故障排查

### 问题1: 构建失败

**检查**:
- Render Dashboard → Logs → Build Logs
- 查找错误信息

**常见原因**:
- 依赖安装失败
- TypeScript编译错误
- 内存不足

**解决**:
```bash
# 本地测试构建
cd backend
npm install
npm run build

# 如果成功，问题可能在Render配置
```

### 问题2: 启动失败

**检查**:
- Render Dashboard → Logs → Runtime Logs
- 查找启动错误

**常见原因**:
- 启动命令错误（必须是 `node dist/index.pgboss.js`）
- 数据库连接失败
- 环境变量缺失

**解决**:
- 检查启动命令
- 验证数据库连接信息
- 确认所有环境变量已配置

### 问题3: Debug信息未显示

**检查**:
- 环境变量 DEBUG_MODE=true
- 服务已重启

**解决**:
```bash
# 1. 在Render Dashboard添加环境变量
DEBUG_MODE=true

# 2. 保存并等待服务重启

# 3. 查看日志确认
```
